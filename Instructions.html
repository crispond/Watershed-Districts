Watershed Districts Instructions

Step 1

Begin with WatershedDistricts.sql. This will list the steps required to recreate district geometries. This process may take several days, depending on the size of the data and the machine processing it. It is theoretically possible to use it with any source dataset containing the applicable elements, and there is also a method by which US congressional districts may be assigned without regard to state boundaries. This results in a far more equitable distribution of population.

Step 2

Download all required data.

Step 3

Install required functions.

Step 4

Run each PGScript batch section in order. It's recommended to run each section separately and to run the included sanity checks afterward, in case data were not imported correctly.

Step 5

Update the database connexion in include.php and run the listed PHP files, in order.

Step 6

If desired, export final shapes as text or KML with the SQL code at the very end of WatershedDistricts.sql.
